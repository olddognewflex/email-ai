# Nest Module Implementation Skill

When implementing a NestJS module in this repository:

1. Read AGENTS.md first.
2. Keep the module focused on a real domain boundary.
3. Add only the necessary:
   - module
   - service/provider
   - controller if externally exposed
   - DTOs only when useful
4. Avoid unnecessary interfaces and wrappers.
5. Keep external integrations isolated.
6. Keep side effects explicit.
7. Include verification steps and note any follow-up work.
